# Sistema de Controle de Estoque

Projeto acadêmico desenvolvido em Python para a disciplina de Engenharia de Software.

O sistema simula um controle simples de estoque, permitindo registrar entrada e saída de produtos, atualizar quantidades automaticamente e consultar o histórico de movimentações.

## Funcionalidades

- Listar produtos cadastrados;
- Registrar entrada de produtos;
- Registrar saída de produtos;
- Validar se há estoque suficiente antes da retirada;
- Registrar movimentações com data e responsável;
- Exibir histórico de entradas e saídas.

## Tecnologias utilizadas

- Python 3

## Como executar

Clone o repositório:
```bash
git clone https://github.com/LeoNT7/Trabalho-Engenharia-De-Software.git
```

## Acesse a pasta do projeto:

cd Trabalho-Engenharia-De-Software

## Execute o arquivo principal:
```bash
python main.py
```
Ou:
```bash
py main.py
```
